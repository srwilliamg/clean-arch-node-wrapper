# API

#### run using docker
>
> npm run docker:build
> npm run docker:start
> npm run db:migrate
> send a GET request to localhost:3000

#### run without docker

> create database using postgres and call it 'postgres'
> npm install
> npm run db:migrate
> send a GET request to localhost:3000

#### Test
>
> npm run test
>
#### Format and linter
>
> npm run format and npm run lint
